# Portfoloi

A modern personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- TypeScript
- Vite 5
- Tailwind CSS 4

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Project Structure

```text
src/
  App.tsx
  main.tsx
  index.css
  components/
    Navbar.tsx
    ProjectModal.tsx
    SectionHeader.tsx
  data/
    portfolio.ts
```

## Notes

- Main page layout and sections are managed in `src/App.tsx`.
- Portfolio content (skills, projects, process, achievements, music) is managed in `src/data/portfolio.ts`.
- Global design tokens and animations are in `src/index.css`.
